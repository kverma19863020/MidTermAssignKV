# MidTermAssignKV
Advanced Python Calculator using multiple design patterns.

## Design Patterns Used

- Factory Pattern
- Memento Pattern (Undo)
- Observer Pattern (Logging)

## Features

- Arithmetic Operations
- CSV History Storage
- Logging System
- Undo Functionality
- Input Validation
- Exception Handling
- Command Line Interface

## Run Application

```bash
python -m app.calculator
